Virtual Capability: advertised_device_os:
Capability infers the name of the device OS based on user-agent string analysis and -possibly- HTTP headers and WURLF capabilities.
Virtual Capability: complete_device_name:
Capability would contatenates brand name, model name and marketing name (Where available) of a device into a single string.
Static Capability: is_tablet:
Capability is set to true if device is a tablet computer such as iPad or similar, regardless of OS.
